# Launcher Wiki

Welcome to the official documentation for **Launcher**, a modern desktop application launcher for Linux built with GTK4 and Adwaita.

## Overview

**Launcher** is a fast, intuitive, and extensible application launcher designed for Linux desktop environments. It provides instant search capabilities, a beautiful user interface, and seamless integration with your installed applications.
